## 2. Установка Git

Загружаем последнюю версию Git с [сайта](https://git-scm.com/downloads)
```
git help -a
git commit -am "Insert #2" 
git commit -a -m "Insert #2" 
git add
git commit -m "Insert #2" 
git commit --amend -m "Message"
git log --oneline
```
